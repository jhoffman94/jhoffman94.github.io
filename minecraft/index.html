SkyFactory 4 Minecraft Server

This page details specific changes that were made to the stock SkyFactory 4 modpack for our server playthrough.

World Changes

The overworld uses the Sky Island world gen. No other changes have been made to other dimensions from SkyFactory 4 defaults.
New players should use the command: /topography island home to have the game assign them an island to build on.

Recipe Changes

Changes to the AE2 processor recipes are balanced to require the same technologies as you would need to create the item through the use of the stock AE2 inscriber recipe.
This is a quality-of-life recipe addition as the AE2 Inscriber is very difficult to sanely automate.
The presses are not consumed as part of the craft and any mod's Silicon can be used.
The left recipe is intended for use in a crafting table - the presses will not be consumed.
The right recipe is intended for AE2 autocrafting - which will require some inscriber automation.
