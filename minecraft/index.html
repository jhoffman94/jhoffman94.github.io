SkyFactory 4 Minecraft Server

This page details specific changes that were made to the stock SkyFactory 4 modpack for our server playthrough.

World Changes

The overworld uses the Sky Island world gen. No other changes have been made to other dimensions from SkyFactory 4 defaults.
New players should use the command: /topography island home to have the game assign them an island to build on.

Config & Recipe Changes

Applied Energistics 2

Changes to the AE2 processor recipes are balanced to require the same technologies as you would need to create the item through the use of the stock AE2 inscriber recipe. This is a quality-of-life recipe addition as the AE2 Inscriber is very difficult to sanely automate.

The left recipe is intended for use in a crafting table - the presses will not be consumed in crafting tables and any mod's Silicon can be used.

The right recipe is intended for AE2 autocrafting - which will require some sane inscriber automation.

Horsepower

The use of horse-powered machines (through the use of a vanilla Lead) has been disabled. This mod's manual machines are still functional.

Vanilla

While Leads are not disabled as the item is used as a crafting material for other mods, their use is strongly discouraged at this time due to unexplained server tickrate lag. Use Fences or Walls to contain horses and animals, rather than hitching posts.

Client-side JEI Recipe Update

This script is not required to play on the server, however, these recipes will only appear correctly in your game if you download the recipe script file.
